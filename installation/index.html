From Source

Requires Rust 1.70+.

# Install from GitHub
cargo install --git https://github.com/rnabioco/aform-rs

# Or clone and build
git clone https://github.com/rnabioco/aform-rs
cd aform-rs
cargo build --release

The binary will be at target/release/aform.
